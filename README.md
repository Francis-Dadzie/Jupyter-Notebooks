# Data Science & Machine Learning Projects

A comprehensive collection of Jupyter Notebook projects demonstrating practical applications of data science and machine learning techniques to real-world datasets.

## 📋 Overview

This repository showcases the complete data science pipeline:
- **Data Exploration & Preparation**: Loading, validating, and preparing raw data
- **Exploratory Data Analysis (EDA)**: Uncovering patterns, trends, and relationships
- **Descriptive Statistics**: Summarizing data characteristics and distributions
- **Time Series Analysis**: Analyzing temporal patterns and forecasting
- **Machine Learning**: Building and evaluating predictive models
- **Data Visualization**: Creating insightful visual representations

---

## 📚 Projects

### 1. **Walmart Sales Data Analysis**
Comprehensive analysis of Walmart store sales performance and factors influencing weekly sales.

**Files:** 
- `Walmart sales data analysis.ipynb`
- `Walmart_Store_sales.csv` (dataset)

**Key Analysis:**
- Dataset: 6,435 records across 45 stores (2010-2012)
- Descriptive statistics: mean, median, mode, quartiles
- Distribution analysis and skewness assessment
- Seasonal trends and sales variability
- Measures of dispersion (variance, standard deviation, coefficient of variation)
- Outlier detection using z-scores
- Impact analysis of economic factors on sales

**Features Analyzed:**
- Store ID, Weekly Sales, Date
- Holiday Flag, Temperature
- Fuel Price, CPI (Consumer Price Index)
- Unemployment Rate

**Key Findings:**
- Average weekly sales: $1,046,965
- Median weekly sales: $960,746
- Sales range: $209,986 to $3,818,686
- No missing values or duplicates
- Positively skewed distribution

## 🚀 Getting Started

### Prerequisites
```bash
pip install pandas numpy matplotlib seaborn scipy scikit-learn plotly jupyter
```

### Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/Francis-Dadzie/Jupyter-Notebooks.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Jupyter-Notebooks
   ```

3. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

4. Open any notebook and run the cells to explore the analysis
